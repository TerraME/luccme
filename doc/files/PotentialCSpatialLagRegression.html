PotentialCSpatialLagRegression

Similar to the LinearRegression approach, but relies spatial regression techniques to estimate the regression cover (considers the spatial dependence of the land use).

Arguments

  • adaptRegressionConstants: Handles with the constants regression method of a PotentialCSpatialLagRegression component.
  • computePotential: Handles with the modify method of a PotentialCSpatialLagRegression component.
  • landUseDrivers: The land use drivers fields in database.
  • modify: Handles with the modify method of a PotentialCSpatialLagRegression component.
  • modifyDriver: Modify potencial for an protected area.
  • potentialData: A table with the regression parameters for each attribute.
  • potentialData.betas: A linear regression betas for land use drivers and the index of landUseDrivers to be used by the regression (attributes).
  • potentialData.const: A linear regression constant.
  • potentialData.isLog: Inform whether the model is part of a coupling model.
  • potentialData.maxReg: A coefficient to potentiality the regression value.
  • potentialData.minReg: A coefficient to minimize the regression value.
  • potentialData.ro: Auto regressive coefficient.
  • run: Handles with the execution method of a PotentialCSpatialLagRegression component.
  • verify: Handles with the verify method of a PotentialCSpatialLagRegression component.

Return value:

    The modified component.
