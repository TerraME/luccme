PotentialDSampleBased

Component developed based on MaxEnt behaviour, considering the attributes values of the samples cells to calculate the potencial of the use on the whole cellular space.

Arguments

  • potentialData: A table with the data need to run the component.
  • potentialData.attributesClass: A list of the select attributes (categorical) to analysis.
  • potentialData.attributesPerc: A list of the select attributes (range) to analysis.
  • potentialData.cellUsePercentage: On discrete components must be 100.

Return value:

    The modified component.

Usage

P1 = PotentialDSampleBased
{
 potentialData =
 {
   -- Region 1
   {
     -- floresta
     {
       cellUsePercentage = 100, 

       attributesPerc = 
       {
         "dist_rodovias",
         "dist_centrourbano",
       },

       attributesClass = 
       {
         "uc_us",
         "fertilidadealtaoumedia",
         "assentamentos",
         "ti",
         "desmatamento"
       }
     },

     -- desmatamento
     {
       cellUsePercentage = 100, 

       attributesPerc = 
       {
         "dist_rodovias",
         "dist_centrourbano",
       },

       attributesClass = 
       {
         "uc_us",
         "fertilidadealtaoumedia",
         "assentamentos",
         "ti",
       }
     },

     -- outros
     {
       cellUsePercentage = 100, 

       attributesPerc = 
       {
       },

       attributesClass = 
       {
       }
     }
   }
 }
}
