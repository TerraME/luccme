PotentialCSpatialLagLinearRegressionMix

Modification of the SpatialLagRegression component to allow the use of simple linear regression in specific cases, when roads are created or paved. The component is an example of how the framework can be extended for specific applications. It was created to allow the representation of the creation new deforestation frontiers in the Brazilian Amazon.

Arguments

  • adaptRegressionConstants: Handles with the constants regression method of a PotentialCSpatialLagLinearRegressionMix component.
  • computePotential: Handles with the modify method of a PotentialCSpatialLagLinearRegressionMix component.
  • landUseDrivers: The land use drivers fields in database.
  • modify: Handles with the modify method of a PotentialCSpatialLagLinearRegressionMix component.
  • modifyDriver: Modify potencial for an protected area.
  • modifyRegression: Handles with the modify regression method of a PotentialCSpatialLagLinearRegressionMix component.
  • potentialData: A table with the regression parameters for each attribute.
  • potentialData.betas: A linear regression betas for land use drivers and the index of landUseDrivers to be used by the regression (attributes).
  • potentialData.const: A linear regression constant.
  • potentialData.isLog: Inform whether the model is part of a coupling model.
  • potentialData.maxReg: A coefficient to potentiality the regression value.
  • potentialData.minReg: A coefficient to minimize the regression value.
  • potentialData.ro: Auto regressive coefficient.
  • run: Handles with the execution method of a PotentialCSpatialLagLinearRegressionMix component.
  • verify: Handles with the verify method of a PotentialCSpatialLagLinearRegressionMix component.

Return value:

    The modified component.
